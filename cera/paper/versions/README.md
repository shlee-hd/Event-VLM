# CERA Paper Versions

Version history for the clean-room CERA manuscript.

## Convention

- Version path: `cera/paper/versions/vN/`
- Minimum files:
  - `main.tex`
  - `main.bib`
  - `NOTES.md`
- Optional:
  - `CERA-paper-vN.pdf`
